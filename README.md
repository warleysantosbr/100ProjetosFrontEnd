## 🧠 Meu Método de Estudo

📌 A ideia aqui **não é decorar código**, e sim **entender o que cada linha faz**. Para isso:

1. **Assisto o vídeo do projeto** no canal AsmrProg.
2. **Copio o código na minha IDE (VS Code)** acompanhando o que é feito.
3. Se **não entendo alguma linha**, venho aqui no **ChatGPT** e **peço uma explicação detalhada**.
4. Ao final, **testo o projeto no navegador** e faço ajustes se necessário.
